## [LeetCode有效的括号](https://leetcode-cn.com/problems/valid-parentheses/)

给定一个只包括 `'('，')'，'{'，'}'，'['，']'` 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:
```
输入: "()"
输出: true
```
示例 2:
```
输入: "()[]{}"
输出: true
```
示例 3:
```
输入: "(]"
输出: false
```
示例 4:
```
输入: "([)]"
输出: false
```
示例 5:
```
输入: "{[]}"
输出: true
```
***
用栈(stack)和表(map)来遍历就可以了
如果是`左边的括号`-入栈`st.push(c)`
否则栈顶作为键找出对应的右括号比较
```
Parenthesis.at(st.top()) != c
```
遍历结束,判断栈是否为空
- 是, 返回`true`
- 否, 返回`false`

优化:
- 字符串长度为奇数必为非法直接返回`false`
